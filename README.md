# Resolvendo Algoritmos com GitHub Copilot 🤖

Este repositório foi criado como parte de um desafio da DIO para explorar as capacidades do **GitHub Copilot** e **GitHub Codespaces**.

## 🛠️ Tecnologias Utilizadas
- **Python**: Linguagem principal.
- **GitHub Copilot**: IA para auxílio na codificação.
- **GitHub Codespaces**: Ambiente de desenvolvimento na nuvem.

## 📋 Desafios Implementados
1. **Concatenar Strings**: Une duas entradas do usuário.
2. **Repetir Textos**: Multiplica uma string pelo número desejado.
3. **Calculadora Simples**: Realiza operações básicas entre dois números.

## 🚀 Como Executar
1. Dê um fork neste repositório.
2. Abra no seu GitHub Codespaces.
3. Execute o comando `python desafios/algoritmos.py`.
