# Resolvendo Códigos em Python com o GitHub Copilot 🤖

Olá! Este repositório foi desenvolvido como parte de um desafio prático da **DIO (Digital Innovation One)**. O objetivo principal é explorar como o **GitHub Copilot** e o **GitHub Codespaces** podem acelerar o desenvolvimento de algoritmos simples em Python, auxiliando na produtividade e documentação.

## 🛠️ Tecnologias Utilizadas
- **Python**: Linguagem de programação utilizada para os desafios.
- **GitHub Copilot**: Ferramenta de IA que auxilia na escrita de código.
- **GitHub Codespaces**: Ambiente de desenvolvimento completo baseado na nuvem.

## 📂 Estrutura do Projeto
Os códigos foram organizados em uma pasta dedicada para facilitar a navegação e o versionamento:

- `desafios/`: Pasta contendo os scripts Python.
  - `concatena_dados.py`: Une dois dados informados pelo usuário.
  - `repetindo_textos.py`: Repete uma string uma quantidade específica de vezes.
  - `operacoes_matematicas.py`: Realiza operações básicas (+, -, *, /) entre dois números.

## 📋 Desafios Implementados

### 1 - Concatenando Dados 🐾
**Descrição:** Recebe dois dados diferentes do usuário e os concatena em uma única string utilizando f-strings ou o operador de soma.
- **O que aprendemos:** Manipulação de strings, entrada de dados (`input`) e concatenação.

### 2 - Repetindo Textos 🖊️
**Descrição:** Solicita uma string e um número inteiro para retornar a string repetida o número de vezes informado.
- **O que aprendemos:** Tipos de dados (int), operadores aritméticos aplicados a strings e sugestões do Copilot para repetição.

### 3 - Operações Matemáticas Simples 📐
**Descrição:** Solicita dois números e realiza uma operação matemática básica escolhida pelo usuário.
- **O que aprendemos:** Operações aritméticas, estruturas condicionais (`if/elif/else`) e tratamento básico de erros (como divisão por zero).

## 🚀 Como Executar
1. Faça um **Fork** deste repositório.
2. No seu repositório forkeado, clique no botão **Code** e abra um **Codespace**.
3. No terminal do Codespaces, você pode executar cada desafio individualmente com os comandos:
   - `python desafios/concatena_dados.py`
   - `python desafios/repetindo_textos.py`
   - `python desafios/operacoes_matematicas.py`
